This repository contains the code for the analyses and figures contained in the "Mapping *trans*-eQTLs at single-cell resolution using Latent
Interaction Variational Inference" manuscript (Vagiaki 2026).

The *Latent Interaction Variational Inference* (LIVI) framework is available at [https://github.com/PMBio/LIVI](https://github.com/PMBio/LIVI)
